This GitHub repository contains all files used in the development, testing and instantiation of the dissertation project:
	
Evaluating the efficacy of anonymised crowd counting in public spaces using Computer Vision and Edge AI solutions


Files are commented as necessary.

This repo contains:
	- .csv files of data collected from model results
		- apart from labels.csv, which was taken from the dataset
	- Jupyter Notebook files used to test and train models
	- .py files for instantiating the system on a Raspberry Pi, taken from the Pi
	- .php, .html, .css, and .js files for the web application.
